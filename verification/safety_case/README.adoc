== Purpose

A Goal-Structuring-Notation (GSN) assurance argument for BetLang's
gambling-harm-reduction guarantees described in `SAFETY-FEATURES.md`:

* *Dutch-book freedom* — coherent probability assignments cannot be
  arbitraged.
* *Risk-of-ruin protection* — bankroll bounds.
* *Cool-off* and *stake limits*.

Each top-level claim is decomposed to sub-goals discharged by the
mechanised proofs (`../proofs/`) and tests (`../tests`,
`../conformance`), with the proof obligations registered in
`../../PROOF-NEEDS.md`.
